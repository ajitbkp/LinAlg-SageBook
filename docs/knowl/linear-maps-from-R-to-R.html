Example 3.1.4.

Let us find all linear maps from \(\R\) to \(\R\text{.}\)

Suppose \(f\colon \R\to \R\) is a linear map. We know that \(\{1\}\) is basis of \(\R\text{.}\) Then any vector \(x=x\cdot 1\text{.}\) Therefore, \(f(x)=f(x\cdot 1)=xf(1)\text{.}\) Thus if we define \(f(1)=a\in \R\) \(f(x)=ax\text{.}\) If \(f\) is linear map from \(\R\to \R\text{,}\) then there exist a real number \(a\in \R\) such that \(f(x)=ax\text{.}\) Note that in this case \(a=f(1)\text{.}\)
