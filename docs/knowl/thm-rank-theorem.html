Theorem 1.4.5.

Suppose \(A\) is an \(m \times n\) matrix. The rank of \(A\) is equal to the number of non-zero rows in an echelon equivalent form of \(A\text{.}\) In other words, the rank of matrix \(A\) is the number of leading 1's in any row-echelon matrix to which \(A\) can be carried by row operations. \qed
