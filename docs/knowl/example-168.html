Example 6.3.4.

Find the orthogonal complement of \(span(\{\left(2,\,0,\,1,\,-1\right), \left(1,\,2,\,0,\,-1\right)\})\text{.}\)

\begin{align*} U^\perp =\amp \{(x_1,x_2,x_3,x_3): 2x_1+x_3-x_4=0,x_1+2x_2-x_4=0\}\\ =\amp null\left(\begin{array}{rrrr} 2 \amp 0 \amp 1 \amp -1 \\ 1 \amp 2 \amp 0 \amp -1 \end{array} \right)\\ =\amp span(\{(1, 0, -1, 1),(0, 1, 2, 2)\}) \end{align*}
