Definition 3.1.1.

A map \(T\colon \R^n \to \R^m\) is called a linear transformation (or linear map) if it satisfies the following properties:

(i) for all \(x,y\in \R^n\text{,}\) \(T(x+y)=T(x)+T(y)\text{.}\)

(ii) for all \(\alpha\in \R\) and \(x\in \R^n\text{,}\) \(T(\alpha x)=\alpha T(x)\) .
