Definition 1.0.2.

A solution of the system of linear equations (1.0.1) is an ordered \(n\)-tuple of the real numbers \((y_1, y_2, \cdots, y_n)\) such that \(x_i = y_i\) for each \(i = 1, 2, \ldots, n\) satisfies each of the \(m\)-equations. The set of all solutions of a system is called the solution set. It is a subset of \(\R^n\text{,}\) the Cartesian product of the set \(\R\) of real numbers \(n\)-times. If we denote the solution set by \(S\subset \R^n\text{,}\) then
\begin{equation*} S=\{(x_1,\ldots,x_n)\in \R^n:\sum_{j=1}^n a_{ij}x_j =b_i,\quad 1\leq i\leq m\}\text{.} \end{equation*}
If the solution set is nonempty then the system is said to be consistent and otherwise it is said to be inconsistent.
