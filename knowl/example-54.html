Example 3.1.2. Examples of linear transformation.

(i) If \(T\colon \R^n\to \R^m\) is a linear map, then \(T(0)=0\text{.}\) That is, \(T\) takes the zero vector in \(\R^n\) to the zero vector in \(\R^m\text{.}\)

(ii) If \(T\colon \R^n\to \R^m\) is a linear map, then \(T(-v)=-T(v)\) for all \(v\in \R^n\text{.}\)
