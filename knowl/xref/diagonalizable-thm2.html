Theorem 5.2.9.

Let \(A\) be an \(n\times n\) real matrix with distinct eigenvalues \(\lambda_1,\ldots, \lambda_k\) and algebraic multiplicity \(m_1,\ldots, m_k\) respectively. Then \(A\) is diagonalizable if and only if algebraic multiplicity is same as geometric multiplicity for each eigenvalue. That is \(m_i=\dim{(E_{\lambda_i})}\) for \(i=1,2,\ldots,k\text{.}\)
