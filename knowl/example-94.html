Example 4.1.10.

Let \(V=(0,\infty)\text{.}\) Define addition and scalar multiplication on \((0,\infty)\) as follows:

\begin{equation*} x+y:=xy \text{ and } \alpha \cdot x:= x^\alpha\text{.} \end{equation*}

Check that \((0,\infty)\) under this addition and scalar multiplication is a vector space over \(\R\text{.}\) Contrast this example with Example Example 4.1.9.
