Theorem 1.4.8. Number of Solution of a System.

Let \(AX = B\) be a system of \(m\) linear equations in \(n\) unknowns and \(A^* = [A~|~B]\text{,}\) the augmented matrix of the system. Then

  1. The system has a solution i.e. the system is consistent if and only if \(r(A) = r(A^*)\text{.}\)

  2. If \(r(A) = r(A^*) \) and \(r(A) = r \) whihc is strictly less than \(n\text{,}\) then number of unknowns, then there are infinite number of solutions and \(n-r\) variables can be chosen freely.

  3. If \(r(A) = r(A^*)\) and \(r(A) = n\text{,}\) the number of unknowns, then there is only one (unique) solution for the system.
